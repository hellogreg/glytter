Glytter: the font glyph getter

This tool lets you view all characters for most font files (.ttf, .otf, .woff, or .woff2) on your local machine. Fonts are kept on your machine and not uploaded anywhere.

Current Font: Firava

Notes

  • View this project on GitHub.
  • Glytter uses fontkit to parse font info.
  • The page's default typeface, Firava, is a variable adaptation of Mozilla's Fira Sans.
  • This project was created by me, Greg Gibson, because I wanted a simple tool for viewing and copying character sets for fonts I produce or use.
